## Getting Started

### Prerequisites

- Node.js 18+ and npm
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/ivi374forivi/public-record-data-scrapper.git
cd public-record-data-scrapper

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
